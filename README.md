# News-Feed
* News feed is an android application that brings
the latest news to the user with the usage of Guardian api.

# Functions Implemented

* AsyncTaskLoader
* ListView
* HttpUrlConnection
* Internet Permission
* Custom ArrayAdapter
* StringBuilder
* JSON Parsing
